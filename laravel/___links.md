## Composer

### Add private repo (with ssh keys)

<https://getcomposer.org/doc/04-schema.md#repositories>

```json
{
    "require": {
        "vendor/my-private-repo": "dev-master"
    },
    "repositories": [
        {
            "type": "vcs",
            "url":  "git@bitbucket.org:vendor/my-private-repo.git"
        }
    ]
}
```
